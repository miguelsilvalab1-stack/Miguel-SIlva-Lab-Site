Política de Privacidade

Última atualização: 6 de fevereiro de 2026

1. Responsável pelo Tratamento

O responsável pelo tratamento dos dados pessoais recolhidos neste website é Miguel Silva Lab, com contacto através do email miguelsilvalab1@gmail.com.

2. Dados Pessoais Recolhidos

Através do formulário de contacto deste website, podemos recolher os seguintes dados pessoais:

  • Nome
  • Endereço de email
  • Mensagem (conteúdo livre)

3. Finalidade do Tratamento

Os dados pessoais são recolhidos com a finalidade de:

  • Responder a pedidos de contacto e informação
  • Apresentar propostas de serviços solicitados
  • Comunicar sobre serviços de consultoria e formação em IA

4. Base Legal

O tratamento dos dados pessoais baseia-se no consentimento do titular, prestado através do envio voluntário do formulário de contacto, nos termos do artigo 6.º, n.º 1, alínea a) do Regulamento Geral sobre a Proteção de Dados (RGPD).

5. Conservação dos Dados

Os dados pessoais são conservados apenas pelo período necessário para a finalidade para a qual foram recolhidos, ou conforme exigido por lei.

6. Direitos do Titular

Nos termos do RGPD, o titular dos dados tem direito a:

  • Aceder aos seus dados pessoais
  • Solicitar a retificação de dados inexatos
  • Solicitar o apagamento dos dados
  • Limitar o tratamento dos dados
  • Portabilidade dos dados
  • Opor-se ao tratamento
  • Retirar o consentimento a qualquer momento

Para exercer qualquer destes direitos, entre em contacto através de miguelsilvalab1@gmail.com.

7. Partilha de Dados

Os dados pessoais não são partilhados com terceiros, exceto quando necessário para o cumprimento de obrigações legais.

8. Segurança

Implementamos medidas técnicas e organizativas adequadas para proteger os dados pessoais contra o acesso não autorizado, a alteração, a divulgação ou a destruição.

9. Cookies

Este website não utiliza cookies de rastreamento. Apenas são utilizados cookies técnicos essenciais para o funcionamento do site, quando aplicável.

10. Alterações

Esta política de privacidade pode ser atualizada periodicamente. Quaisquer alterações serão publicadas nesta página com a data de atualização.

11. Contacto

Para questões relacionadas com a proteção de dados pessoais, contacte-nos através de miguelsilvalab1@gmail.com.
